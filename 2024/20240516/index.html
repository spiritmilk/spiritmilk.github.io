MambaOut-Do We Really Need Mamba for Vision?

mambaжҳҜз»§transformerд№ӢеҗҺеӨ§зҒ«зҡ„з»“жһ„д№ӢдёҖгҖӮд№ҹж¶ҢзҺ°дәҶеҗ„з§ҚmambaпјҢеҗ„з§ҚйўҶеҹҹзҡ„mambaгҖӮжң¬еҚҡе®ўд№ӢеүҚд№ҹд»Ӣз»ҚдәҶиҝҷдёҖз®—жі•гҖӮ

еҮәиҮӘNUSзҡ„Weihao Yu, Xinchao WangзӯүдәәжҸҗеҮәдәҶзәҜеҚ·з§Ҝзҡ„mamaoutж„Ҹеӣҫжү“иҙҘmambaгҖӮ

MambaOut жЁЎеһӢеңЁ ImageNet еӣҫеғҸеҲҶзұ»дёҠи¶…и¶ҠдәҶжүҖжңүи§Ҷи§ү Mamba жЁЎеһӢпјҢиЎЁжҳҺ Mamba еҜ№дәҺиҜҘд»»еҠЎзЎ®е®һжҳҜдёҚеҝ…иҰҒзҡ„гҖӮиҮідәҺжЈҖжөӢе’ҢеҲҶеүІпјҢMambaOut ж— жі•дёҺжңҖе…Ҳиҝӣзҡ„и§Ҷи§ү Mamba жЁЎеһӢзҡ„жҖ§иғҪзӣёеӘІзҫҺпјҢиҝҷеұ•зӨәдәҶ Mamba еңЁй•ҝеәҸеҲ—и§Ҷи§үд»»еҠЎдёӯзҡ„жҪңеҠӣгҖӮ

д»Һи®°еҝҶи§’еәҰжқҘзңӢеӣ жһңжіЁж„ҸеҠӣе’Ңзұ» RNN жЁЎеһӢзҡ„жңәеҲ¶иҜҙжҳҺпјҢе…¶дёӯ$x_i$ иЎЁзӨәз¬¬ i жӯҘйӘӨзҡ„иҫ“е…Ҙж Үи®°гҖӮ (a) еӣ жһңжіЁж„ҸеҠӣе°ҶжүҖжңүе…ҲеүҚж Үи®°зҡ„й”® е’ҢеҖј еӯҳеӮЁдёәеҶ…еӯҳгҖӮйҖҡиҝҮдёҚж–ӯж·»еҠ еҪ“еүҚ token зҡ„ key е’Ң value жқҘжӣҙж–°еҶ…еӯҳпјҢеӣ жӯӨеҶ…еӯҳжҳҜж— жҚҹзҡ„пјҢдҪҶзјәзӮ№жҳҜйҡҸзқҖеәҸеҲ—зҡ„е»¶й•ҝпјҢж•ҙеҗҲж—§еҶ…еӯҳе’ҢеҪ“еүҚ token зҡ„и®Ўз®—еӨҚжқӮеәҰдјҡеўһеҠ гҖӮеӣ жӯӨпјҢжіЁж„ҸеҠӣеҸҜд»Ҙжңүж•Ҳең°з®ЎзҗҶзҹӯеәҸеҲ—пјҢдҪҶеҸҜиғҪдјҡйҒҮеҲ°иҫғй•ҝеәҸеҲ—зҡ„еӣ°йҡҫгҖӮ

зӣёеҸҚпјҢзұ»дјј RNN зҡ„жЁЎеһӢе°Ҷе…ҲеүҚзҡ„ж Үи®°еҺӢзј©дёәеӣәе®ҡеӨ§е°Ҹзҡ„йҡҗи—ҸзҠ¶жҖҒ в„Һ пјҢз”ЁдҪңеҶ…еӯҳгҖӮиҝҷз§Қеӣәе®ҡеӨ§е°Ҹж„Ҹе‘ізқҖ RNN еҶ…еӯҳжң¬иҙЁдёҠжҳҜжңүжҚҹзҡ„пјҢж— жі•дёҺжіЁж„ҸеҠӣжЁЎеһӢзҡ„ж— жҚҹеҶ…еӯҳе®№йҮҸзӣҙжҺҘз«һдәүгҖӮе°Ҫз®ЎеҰӮжӯӨпјҢзұ»дјј RNN зҡ„жЁЎеһӢеңЁеӨ„зҗҶй•ҝеәҸеҲ—ж—¶еҸҜд»ҘиЎЁзҺ°еҮәжҳҺжҳҫзҡ„дјҳеҠҝпјҢеӣ дёәж— и®әеәҸеҲ—й•ҝеәҰеҰӮдҪ•пјҢе°Ҷж—§еҶ…еӯҳдёҺеҪ“еүҚиҫ“е…ҘеҗҲе№¶зҡ„еӨҚжқӮжҖ§дҝқжҢҒдёҚеҸҳгҖӮ

жҖ»д№ӢпјҢMamba йқһеёёйҖӮеҗҲе…·жңүд»ҘдёӢзү№еҫҒзҡ„д»»еҠЎпјҡ
вҖў зү№еҫҒ1пјҡд»»еҠЎж¶үеҸҠеӨ„зҗҶй•ҝеәҸеҲ—гҖӮ
вҖў зү№еҫҒ2пјҡд»»еҠЎйңҖиҰҒеӣ жһңж Үи®°ж··еҗҲжЁЎејҸгҖӮ

жҳҜеҗҰйҖӮеҗҲ

жҺҘдёӢжқҘпјҢи®әж–Үе°Ҷи®Ёи®әи§Ҷи§үиҜҶеҲ«д»»еҠЎжҳҜеҗҰиЎЁзҺ°еҮәиҝҷдёӨдёӘзү№еҫҒгҖӮ

и§Ҷи§үиҜҶеҲ«д»»еҠЎзҡ„еәҸеҲ—жҳҜеҗҰеҫҲй•ҝ

еҜ№дәҺImageNetдёҠзҡ„еӣҫеғҸеҲҶзұ»пјҢе…ёеһӢзҡ„иҫ“е…ҘеӣҫеғҸеӨ§е°Ҹдёә 224$^{2}$,д»ҺиҖҢдә§з”ҹеқ—еӨ§е°Ҹдёә 16$^{2}$зҡ„ 14$^{2}=196$ж Үи®°гҖӮжҳҫз„¶пјҢ196 иҝңе°ҸдәҺ$\tau_\mathrm{small}$е’Ң$\tau_\mathrm{base}$ ,иЎЁжҳҺ ImageNet дёҠзҡ„еӣҫеғҸеҲҶзұ»дёҚз¬ҰеҗҲй•ҝеәҸеҲ—д»»еҠЎзҡ„жқЎд»¶гҖӮ

еҜ№дәҺ COCO дёҠзҡ„еҜ№иұЎжЈҖжөӢе’Ңе®һдҫӢеҲҶеүІпјҢжҺЁзҗҶеӣҫеғҸеӨ§е°Ҹдёә$800\times1280$,еҜ№дәҺ ADE2oK дёҠзҡ„иҜӯд№үеҲҶеүІпјҢжҺЁзҗҶеӣҫеғҸеӨ§е°Ҹдёә$512\times2048$ ,д»ӨзүҢж•°йҮҸзәҰдёә4KпјҢз»ҷе®ҡиЎҘдёҒеӨ§е°Ҹ$16^2$гҖӮд»Һ$4K>\tau_\mathrm{small}$е’Ң$4K\approx\tau_\mathrm{base}$ејҖе§ӢпјҢCOCOдёҠзҡ„жЈҖжөӢе’ҢADE2oKдёҠзҡ„еҲҶеүІйғҪеҸҜд»Ҙиў«и®ӨдёәжҳҜй•ҝеәҸеҲ—д»»еҠЎгҖӮ

и§Ҷи§үиҜҶеҲ«д»»еҠЎйңҖиҰҒеӣ жһңж Үи®°ж··еҗҲжЁЎејҸеҗ—

дҪңиҖ…жҢҮеҮәпјҢе®Ңе…ЁеҸҜи§Ғзҡ„д»ӨзүҢж··еҗҲжЁЎејҸе…Ғи®ёдёҚеҸ—йҷҗеҲ¶зҡ„ж··еҗҲиҢғеӣҙпјҢиҖҢеӣ жһңжЁЎејҸеҲҷйҷҗеҲ¶еҪ“еүҚд»ӨзүҢеҸӘиғҪи®ҝй—®е…ҲеүҚд»ӨзүҢзҡ„дҝЎжҒҜгҖӮи§Ҷи§үиҜҶеҲ«иў«еҪ’зұ»дёәзҗҶи§Јд»»еҠЎпјҢе…¶дёӯжЁЎеһӢеҸҜд»Ҙз«ӢеҚізңӢеҲ°ж•ҙдёӘеӣҫеғҸпјҢд»ҺиҖҢж¶ҲйҷӨдәҶеҜ№д»ӨзүҢж··еҗҲзҡ„йҷҗеҲ¶гҖӮеҜ№д»ӨзүҢж··еҗҲж–ҪеҠ йўқеӨ–зҡ„йҷҗеҲ¶еҸҜиғҪдјҡйҷҚдҪҺжЁЎеһӢжҖ§иғҪгҖӮ
