来自斯坦福轮回赛的有趣积分题目

另法： I(a)=∫0πln⁡(1−2acos⁡(x)+a2) dx,   a>1I(a)=\int_{0}^{\pi}\ln\left(1-2a\cos(x)+a^2\right)\ \text{d}x,\;\ a>1I(a)=∫0π​ln(1−2acos(x)+a2) dx, a>1

设三角形ABC，边a,b,ca,b,ca,b,c所对的角为α,β,γ\alpha,\beta,\gammaα,β,γ

根据余弦定理有： c2=a2+b2−2abcos⁡(γ)c^2=a^2+b^2-2ab\cos(\gamma)c2=a2+b2−2abcos(γ)

令b=1,γ=xb=1,\gamma=xb=1,γ=x

则：

$I(a) = \int_{0}^{\pi}\ln\left(c^2\right)\ \text{d}\gamma \ = 2\int_{0}^{\pi}\ln\left(c\right)\ \text{d}\gamma $

根据正弦定理有： sin⁡(α)a=sin⁡(β)b=sin⁡(γ)c\frac{\sin(\alpha)}{a}=\frac{\sin(\beta)}{b}=\frac{\sin(\gamma)}{c}asin(α)​=bsin(β)​=csin(γ)​

故：

$I(a) = 2\int_{0}^{\pi}\ln\left(a\ \frac{\sin(\gamma)}{\sin(\alpha)}\right)\ \text{d}\gamma \ = 2\pi \ln(a)+2\int_{0}^{\pi}\ln\left(\sin(\gamma)\right)\ \text{d}\gamma-2\int_{0}^{\pi}\ln\left(\sin(\alpha)\right)\ \text{d}\gamma $

又因为：γ=π−α−β\gamma=\pi-\alpha-\betaγ=π−α−β,dγ=−dα−dβ\text{d}\gamma=-\text{d}\alpha-\text{d}\betadγ=−dα−dβ

当γ→0:α→π,β→0\gamma\rightarrow 0:\alpha\rightarrow \pi,\beta\rightarrow 0γ→0:α→π,β→0

当γ→π:α→0,β→0\gamma\rightarrow \pi:\alpha\rightarrow 0,\beta\rightarrow 0γ→π:α→0,β→0

故 I(a)=2πln⁡(a)+2∫0πln⁡(sin⁡(γ)) dγ−2[∫π0ln⁡(sin⁡(α))(−dα)+∫00ln⁡(sin⁡(α))(−dβ)]=2πln⁡(a)+2∫0πln⁡(sin⁡(γ)) dγ−2∫0πln⁡(sin⁡(α)) dα=2πln⁡(a)I(a) = 2\pi \ln(a)+2\int_{0}^{\pi}\ln\left(\sin(\gamma)\right)\ \text{d}\gamma\\ -2\left[\int_{\pi}^{0}\ln\left(\sin(\alpha)\right)(-\text{d}\alpha)+\int_{0}^{0}\ln\left(\sin(\alpha)\right)(-\text{d}\beta)\right] \\ = 2\pi \ln(a)+2\int_{0}^{\pi}\ln\left(\sin(\gamma)\right)\ \text{d}\gamma-2\int_{0}^{\pi}\ln\left(\sin(\alpha)\right)\ \text{d}\alpha = 2\pi \ln(a)I(a)=2πln(a)+2∫0π​ln(sin(γ)) dγ−2[∫π0​ln(sin(α))(−dα)+∫00​ln(sin(α))(−dβ)]=2πln(a)+2∫0π​ln(sin(γ)) dγ−2∫0π​ln(sin(α)) dα=2πln(a)
