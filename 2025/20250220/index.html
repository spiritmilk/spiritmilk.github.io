ж— еҒҸе°Ҹж–№е·®KL

иҝҷйҮҢдҪҝз”Ёзҡ„е№¶дёҚжҳҜдј з»ҹзҡ„KLпјҢиҖҢжҳҜдҪҝз”ЁдәҶдёҖдёӘж— еҒҸе°Ҹж–№е·®KLпјҢеҚі$KL=rlogr-(r-1)$

дј з»ҹзҡ„KLпјҡ
$$
K L [ q, p ]=\sum_{x} q ( x ) \operatorname{log} \frac{q ( x )} {p ( x )}=E_{x \sim q} [ \operatorname{log} \frac{q ( x )} {p ( x )} ]
$$
еҰӮдҪ•жһ„е»әдёҖдёӘеҘҪзҡ„дј°и®ЎпјҹдёҖдёӘеҘҪзҡ„дј°и®ЎйҮҸжҳҜж— еҒҸзҡ„пјҲе®ғжңүжӯЈзЎ®зҡ„еқҮеҖјпјүе№¶дё”е…·жңүдҪҺж–№е·®гҖӮ

дёҖдёӘж— еҒҸдј°и®ЎйҮҸk1жҳҜ$k1=log\frac{q(x)}{p(x)}=-log r$пјҢдҪҶд»–жңүй«ҳж–№е·®пјҢеӣ дёәд»–еҜ№дәҺеӣ дёәе®ғеҜ№дәҺдёҖеҚҠзҡ„ж ·жң¬жҳҜиҙҹзҡ„пјҢиҖҢ KL жҖ»жҳҜжӯЈзҡ„гҖӮ

еҸҰдёҖдёӘдҪҺж–№е·®дҪҶеӯҳеңЁеҒҸе·®зҡ„дј°и®ЎйҮҸk2жҳҜ$k2=\frac{1}{2}(log\frac{q(x)}{p(x)})^2=\frac{1}{2}(logr)^2$гҖӮ

е®ғзҡ„жңҹжңӣеҸҜд»Ҙд»Һf-divergenceдёҠиҖғиҷ‘пјҢе…¶е®ҡд№үдёәеҮёеҮҪж•°fзҡ„$D_{f} ( p, q )=E_{x \sim q} [ f ( \frac{p ( x )} {q ( x )} ) ] $гҖӮ

иҖҢ
$$
D_{f} ( p_{0}, p_{\theta} )=\frac{f^{\prime\prime} ( 1 )} {2} \theta^{T} F \theta+O ( \theta^{3} )
$$
е…¶дёӯFжҳҜ$p_\theta=p_0$еӨ„зҡ„Fisher information matrixгҖӮ

$E_{q} [ k_{2} ]=E_{q} [ \frac{1} {2} ( \operatorname{log} r )^{2} ]$пјҢиҖҢ$\frac{1}{2}(log(x))^2$е’Ңдј з»ҹKLеҜ№еә”зҡ„$-log(x)$е…·жңүзӣёеҗҢзҡ„$fвҖҷ'(1)=1$гҖӮ
