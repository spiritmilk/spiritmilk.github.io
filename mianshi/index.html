谷尚 python 初级工程师

（2024年2月27日）

岗位要求：
熟悉基本的python语法
熟悉基本的数据处理numpy,opencv,pyqt5,深度学习框架
主要工作内容：
1，完成公司内部模型验证和测试。
2，维护公司内部数据标注软件和深度学习训练工具。
3. 负责智能硬件平台上的python应用开发工作
4. 参与服务器api接口pyton自动化测试的分析讨论
5. 项目难度不是很大，但需要认真和负责。

未读无回复。

湾影科技 软件研发实习生

（2024年2月27日）

岗位职责：
1、在课题组组长指导下开展科研工作，完成科研课题。
2、协助完成论文、专利等。
应聘条件：
1、生物医学工程、物理、电子、计算机、自动化、机械、医学相关背景，本科/硕士/博士在读学生。
2、有R/Matlab/Python等编程经验或硬件开发经验、英语水平良好者优先；
3、工作踏实认真，思维活跃，态度积极向上，执行力强，有快速学习能力。
待遇条件：
1、具体待遇面议。
2、工作表现优秀者，根据本人意愿，课题组可协助其申请及攻读博/硕士学位。

主要进行了boss直聘沟通，询问了毕业去向（读研or工作），询问了”你目前想学习的方向是什么”，回答“机器学习和大模型”。

领寻 Python实习生

【岗位职责】
\1. 负责内部新项目prototype的开发。（Generative AI/Computer Vision/Marketing Automation等方向）
\2. 部署、使用并评测目前所有的开源或闭源AI Agents（AutoGPT、AgentGPT、GodMode、BabyAGI、Microsoft Jarvis、Camel等）
【岗位要求】
\1. 计算机科学 或 人工智能 或 软件工程类本科及研究生在读；
\2. 热爱计算机，不惧怕学习新技术；
\3. 熟练掌握Python；
\4. 使用过ChatGPT或Claude或其他LLMs；
\5. 强大的责任心及执行力。

（2024年3月1日）

进行了电话面试，主要核实简历真假，要求叙述经历，询问了毕业去向（读研or工作），询问薪资要求与转正计划，询问是否有拿到offer。

看小红书上尽可能不要说0 offer，而是说“如有”，大概就是说“有一两个，在谈薪资”，不能说太多，表现地很海王。

面试后要求了项目的github地址和成绩单。
